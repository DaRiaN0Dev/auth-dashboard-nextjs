# Auth Dashboard (Next.js)

Modern authentication dashboard built with Next.js, TypeScript, TailwindCSS, React Hook Form, Zod, Zustand, and a NestJS authentication backend.

## Features

- Sign In
- Sign Up
- Email Verification
- Forgot Password
- Reset Password
- Session Persistence
- Token Refresh Handling
- Protected Routes
- Form Validation
- Responsive Design
- Reusable UI Components
- Type-Safe API Integration

---

## Tech Stack

- Next.js
- React
- TypeScript
- TailwindCSS
- Zustand
- React Hook Form
- Zod

## Authentication Flow

- User Registration
- User Login
